## 📐 3단계 구조

| 단계 | 구분 | 내용 |
|------|------|------|

| STEP 2 | **중주제 — 문체 선택** | 원하는 세계관·분위기에 맞는 문체를 고른다. 총 11개 장르 |

## ⚖️ 기법 적정 개수 가이드

> **황금 공식:** 뼈대 기법 1개 + 씬 목적 기법 1~2개 = 씬 1개 완성
> 기법이 보이지 않아야 좋은 씬이다. 기법은 감정을 전달하는 도구이지, 목적이 아니다.

| 적용 단위 | 권장 기법 수 | 설명 |
|-----------|------------|------|
| 씬 (Scene) 1개 | **2~3개** | 뼈대 기법 1개 + 살 붙이는 기법 1~2개 |
| 챕터 전체 | **3~5개** | 일관성 있는 기법군 유지 |
| 전체 작품 | **5~8개** | 핵심 시그니처 고정 후 변주만 추가 |
| 감정 고조 씬 | **2개 (최소)** | 기법을 줄이고 리듬·문장 길이로 긴장 조절 |
| 클라이맥스 | **1~2개** | 기법이 눈에 띄면 안 된다. 감정만 남긴다 |

### 클라이맥스 씬 전용 지침

> 클라이맥스는 기법을 쓰는 곳이 아니라 기법이 사라지는 곳이다.

| 단계 | 지침 |
|------|------|
| 문장 길이 | 점점 짧아진다. 마지막 1~3문장이 가장 짧다 |
| 감각 | 한 가지 감각만 남긴다. 시각 또는 촉각 중 택일 |
| 내면 독백 | 끊긴다. 단편적 단어·이미지만 허용 |
| 서술 속도 | Time Expansion — 2~3초를 1페이지로 늘린다 |
| 금지 | 감정 설명, 비유 과잉, 새로운 정보 투입 |
| 남기는 것 | 행동 하나, 감각 하나, 침묵 하나 |

**클라이맥스 권장 기법 조합 (장르별)**

| 장르 | 권장 조합 | 이유 |
|------|-----------|------|
| 일상·힐링 | `Show Don't Tell + Strategic Silence` | 말 없는 행동이 고백보다 강하다 |
| 로판·현판·아포칼 | `Sensory Anchoring + Hard Cut` | 감각 한 방 + 여운 차단이 심박수를 올린다 |
| 느와르·공포 | `Hard Cut + Strategic Omission` | 설명하지 않는 것이 공포·감정 모두를 완성한다 |
| 동양판타·새드엔딩 | `Strategic Omission + Resonant Closure` | 여백이 비극과 감동을 동시에 담는다 |
| 회귀·BL/GL | `Unreliable Narrator lite + Hard Cut` | 부정이 깨지는 순간을 끊어낸다 |

## 🛠️ 공통 기법 전체 목록

> 어느 장르에나 적용 가능한 기법 풀. 뼈대 기법과 살 붙이는 기법 모두 여기서 선택한다.
> 장르 섹션의 기법 표에서는 **이 장르에서의 작동 방식**만 기술한다. 기본 정의는 이 목록을 참조.

| 기법 명칭 (EN) | 정의 | 예시 |
|----------------|------|------|
| **Show Don't Tell** | 감정을 진술하지 않고 행동·신체 반응·감각으로만 표현 | *'설레었다' → '입안이 바짝 말랐다. 심장이 한 박자 건너뛰었다.'* |
| **Free Indirect Discourse (자유간접화법)** | 서술자와 주인공의 목소리를 인용 부호 없이 서술에 뒤섞기 | *'그가 웃었다. 왜 웃는 거지. 저 사람은 항상 그렇게 웃어서—'* |
| **Sensory Anchoring (감각적 앵커링)** | 장면 첫 문장을 반드시 오감 중 하나로 시작해 독자를 즉시 장면 안으로 끌어당김 | *'지하철 환기구에서 찬바람이 올라왔다. 그가 거기 서 있었다.'* |
| **Strategic Silence (침묵 연출)** | 대사 사이에 행동 묘사를 삽입해 말보다 침묵이 더 많은 것을 말하게 함 | *'괜찮아.' 그는 대답 대신 나를 봤다. 한참을.* |
| **Narrative Delay (지연)** | 독자가 기다리는 정보(이름·고백·신원·진실)를 의도적으로 늦게 공개 | *이름 공개를 미루기, 고백 직전 씬 전환* |
| **Push-Pull Dynamic (밀당 구조)** | 가까워지면 밀고, 멀어지면 당기는 관계 긴장감을 전체 구조로 설계 | *한쪽이 무너지면 반드시 다른 쪽이 물러서야 긴장 유지* |
| **Surrogate Confession (대리 고백)** | 직접 고백 없이 행동·사물·제3자를 통해 감정을 전달 | *외투를 걸쳐주기, 먼저 문을 닫아주기, 이름을 나지막이 부르기* |
| **Register Contrast (관계 낙차)** | 공식 석상의 냉담함 ↔ 단둘이 있을 때의 균열. 이 간격이 독자 몰입도를 지배 | *황제 앞에서 예의 → 복도에서 단둘이 = 전혀 다른 사람* |
| **Resonant Closure (여운 종결)** | 챕터를 완전히 닫지 않고 툭 잘린 한 문장으로 끊어 여운을 남김 | *'그는 가지 않았다. 나도 말하지 않았다.'* |
| **Telling Detail (디테일의 온도)** | 설명 없이 캐릭터를 드러내는 구체적 사물·습관·행동 묘사 | *지갑이 낡았다 / 볼펜 뚜껑을 물어뜯는다 / 항상 왼쪽 주머니에 손을 넣는다* |
| **Somatization (신체화)** | 특정 감정을 특정 신체 부위와 반복 연결해 독자가 무의식적으로 패턴 학습 | *흔들릴 때마다 귀 뒤가 뜨거워짐 / 거짓말할 때마다 손가락이 구부러짐* |
| **Unreliable Narrator lite (불신 내레이터 약용)** | 주인공이 자기 감정을 스스로 부정 → 독자만 진실을 알게 만들어 우월감+공감 동시 자극 | *'좋아하는 게 아니었다. 그냥 신경 쓰일 뿐이었다. 그게 다였다.'* |
| **Pathetic Fallacy (환경 투영)** | 자연환경·날씨가 인물 감정을 선행하여 반영. 의도적 감정 코딩 | *첫 만남=봄, 갈등=폭우, 화해=첫눈, 이별=단풍* |
| **Emotional Whiplash (낙차 구조)** | 아름다운 장면 → 갑작스러운 위기로 전환해 독자 심박수를 의도적으로 조절 | *로맨스 절정 직후 외부 위협 등장* |
| **Cliffhanger** | 챕터를 반전·미완·위기·폭로로 끊어 독자가 다음 편을 열게 만드는 갈고리 | *고백 직전 씬 전환, 신원 폭로, 새로운 위협 등장* |
| **In Medias Res** | 사건 한가운데서 시작. 설명 없이 긴장 상황으로 바로 진입 | *'칼이 목에 닿기 직전, 그가 내 손목을 잡았다.'* |
| **Drip-feeding Exposition (세계관 흘리기)** | 세계관·설정 설명을 한꺼번에 쏟지 않고 필요한 순간 조금씩 흘리기 | *대화·행동·반응 속에 자연스럽게 설정 녹이기* |
| **Iceberg Theory (빙산 이론)** | 텍스트에는 10%만 드러내고 90%는 작가가 알고 있기. 말하지 않는 것이 힘 | *세계관·과거·감정의 90%는 작가만 안다* |
| **Chekhov's Gun (체호프의 총)** | 앞에서 무심히 배치한 소품·설정이 후반에 결정적으로 작동 | *독자가 눈치채지 못하게 자연스럽게 심어야 효과* |
| **Parallelism (대구와 반복)** | 같은 문장 구조를 반복해 운율을 만들고 감정을 증폭 | *'꽃이 피면 그가 왔다. 꽃이 지면 그가 떠났다.'* |
| **Synesthetic Setting (공감각 묘사)** | 시각 세계를 청각·후각·촉각과 교차해 장면을 입체적으로 구성 | *'달빛이 차가웠다. 그 안에서 거문고 소리가 천천히 녹았다.'* |
| **Hard Cut (단호한 종결)** | 문장을 감정 설명 없이 끊는다. 독자가 채우게 둔다 | *'그는 떠났다.' — 그뿐. 이후 아무것도 없다.* |
| **Strategic Omission (여백의 미)** | 장면을 끝까지 설명하지 않는다. 독자가 스스로 채우는 공간이 힘이다 | *'그가 손을 뻗었다. 그뿐이었다.'* |
| **Defamiliarization (낯선 형용사)** | 익숙한 사물을 낯설고 정확한 형용사로 묘사해 세계의 질감을 바꿈 | *'차갑다' → '금속 같은' / '조용하다' → '납처럼 무거운'* |
| **Title-to-Name Arc (호칭 변화 서사)** | 직책·성 → 그대 → 이름 호칭 변화를 관계 발전의 타임라인으로 설계 | *이름을 처음 부르는 장면을 이 장르의 가장 강력한 고백으로 설계* |
| **Epistolary Technique (편지·기록 기법)** | 일기·편지·문자·메모 등 기록 매체를 서사에 삽입해 감정의 밀도 높임 | *보내지 못한 편지, 지워진 문자, 오래된 일기장* |
| **Dramatic Irony (극적 아이러니)** | 독자가 등장인물보다 더 많은 정보를 알고 있는 상태를 설계 | *독자는 그가 죽을 운명임을 알지만 주인공은 모른다* |
| **Anaphora (수미상관·반복 첫머리)** | 문장이나 단락의 첫머리를 반복해 감정을 점층적으로 쌓아 올림 | *'그가 웃으면 — 그가 웃으면 — 그가 웃으면 나는 항상—'* |
| **Time Compression / Expansion (시간 조절)** | 감정 고조 씬은 느리게 확장하고, 이동·설명 씬은 빠르게 압축 | *키스 장면 3페이지 / 3일간의 이동 1문장* |
| **Interior Monologue (내면 독백 스트림)** | 의식의 흐름처럼 끊김 없이 이어지는 내면 독백. 주인공 내면의 혼란과 감정을 날것으로 표현 | *'아니야 아니야 아니야 근데 왜 — 아니라고 했잖아 — 근데 왜 웃었어 —'* |
| **Symbolism (상징)** | 특정 사물·색·날씨를 감정이나 관계의 상징으로 반복 설계 | *붉은 실 = 인연 / 깨진 거울 = 자아 분열 / 비 = 고백 전조* |
| **Contrast (대조)** | 두 인물·두 장면·두 감정을 의도적으로 병치해 각각의 강도를 높임 | *차갑던 그가 처음으로 따뜻해지는 순간을 이전 장면과 대조* |
| **Gaze as Event (시선 집중)** | 시선이 대사·행동을 대신한다. 눈이 머무는 곳, 눈을 피하는 순간이 감정의 진술이 된다 | *그가 먼저 눈을 내렸다 / 내내 보다가 내가 돌아보는 순간 시선을 거뒀다* |
| **Cynical Gaze (냉소적 관찰)** | 화자가 세계와 타인을 거리를 두고 관찰한다. 냉소의 균열이 감정 진입점이 된다 | *'그는 선한 사람이었다. 그래서 위험했다.'* |
| **Embodied Exhaustion (신체의 피로)** | 감정 대신 신체의 소진 상태를 서술한다. 싸움·슬픔·긴장을 피로·통증·무게감으로 치환 | *'눈물이 나지 않았다. 어깨가 무거울 뿐이었다.'* |
| **Paradoxical Tenderness (역설적 애정)** | 폭력적·위험한 존재가 보이는 한순간의 부드러움. 그 낙차가 감정을 폭발시킨다 | *총을 든 손이 내 머리카락을 쓸었다. 조심스럽게.* |
| **Noir Atmosphere (빛과 색의 질감)** | 빛의 방향·색의 채도·그림자의 위치로 긴장과 감정을 동시에 설계 | *형광등 하나만 켜진 방 / 빗속에서 담배 연기가 가로등 불빛에 걸렸다* |
| **Symbolic Setting (상징적 배경)** | 배경 자체가 인물의 감정·관계·결말을 상징한다. 무대가 서사를 설명한다 | *폐허 = 회복 불가능한 관계 / 좁은 지하 = 둘만의 세계 / 탁 트인 고지 = 자유와 상실* |

# 📚 중주제 — 장르별 문체 가이드

> **장르 섹션 읽는 법**
> - 기법 정의는 위의 **공통 기법 전체 목록**을 참조
> - 장르 섹션의 "이 문체에서의 작동 방식"은 **같은 기법이 이 장르에서 어떻게 다르게 쓰이는가**를 기술
> - `Surrogate Confession`, `Show Don't Tell`, `Contrast`는 전 장르 사용 가능하나 장르마다 표현 방식이 달라야 한다 → 각 장르의 작동 방식 설명을 반드시 확인

## 01. 일상 로맨스
### *Everyday Romance — 평범한 하루 안에서 피어나는 감정*

> **주요 플랫폼·레퍼런스:** 네이버시리즈 현대로맨스 / 리디 로맨스
> **뼈대 기법 (고정):** `Show Don't Tell + Free Indirect Discourse`

### 기준 작가 & 문체 레퍼런스

**하세쿠라 이스나 (羊山イスナ)** — 「늑대와 향신료」
일상의 교환(건네는 손, 머뭇거리는 침묵)이 로맨스의 본체. 직접 고백 없이 관계의 온도를 올린다.

**김애란** — 「달려라, 아비」 「비행운」
짧고 건조한 문장 사이에 감각의 온기를 집어넣는다. 문장 길이의 변화가 감정 리듬.

**신경숙** — 「엄마를 부탁해」
독자가 주인공 몸 안에 들어와 있는 것처럼. 감정 동사 대신 신체 반응을 서술.

> **예시 문단**
>
> "나 그거 좋아하는데." 그가 말했다.
> "알아." 나도 알고 있었다. 그래서 샀다.
> 그는 잠시 침묵했다. "...고마워."
> — 설명 없이 관계가 보이는 하세쿠라식 대화 구조

### 문체 뼈대 원칙

- 시점: 1인칭 주인공
- 문장 호흡: 짧은 문장 2~3개 → 한 번 길게 호흡
- 감정 서술 금지: '설레었다' 대신 신체 반응으로 대체
- 대화: 자연스럽게 끊기고, 말줄임표·단답형으로 여백 남기기
- 챕터 끝: 툭 잘린 한 문장 — 독자가 페이지를 넘기게 만드는 갈고리

### 소주제 — 기법 목록

**◆ 뼈대 기법 (씬마다 반드시 1개 선택, 전체 작품에서 1~2개 고정)**

| 기법 명칭 | 이 장르에서의 작동 방식 |
|-----------|----------------------|
| **Show Don't Tell** | 감정이 아닌 일상 행동에 감정이 배어있다. '설레었다' 대신 '커피를 세 번 저었다' |
| **Free Indirect Discourse** | 주인공의 생각이 서술에 스며든다. 독자가 주인공과 같은 속도로 감정을 발견한다 |
| **Sensory Anchoring** | 일상 감각(냄새·소음·체온)으로 시작. 특별한 배경이 아니라 익숙한 감각이 감정을 연다 |

**◆ 살 붙이는 기법 (씬 목적에 따라 1~2개 선택)**

| 기법 명칭 | 이 장르에서의 작동 방식 |
|-----------|----------------------|
| **Somatization** | 이 장르만의 신체화 신호를 하나 고정한다. 작품 전체에서 반복해 독자가 무의식적으로 학습하게 |
| **Unreliable Narrator lite** | 일상 속 부정. "그냥 습관이야"가 쌓여 감정이 된다. 독자는 알지만 주인공은 끝까지 부정 |
| **Telling Detail** | 캐릭터의 계급·성격·상처를 소품 하나로 드러낸다. 설명 없이 인물이 보인다 |
| **Strategic Silence** | 대화 사이의 행동이 대사보다 많은 것을 말한다. 침묵이 길수록 감정이 높다 |
| **Resonant Closure** | 챕터 마지막 한 문장. 닫지 않는다. 독자가 그 여운을 들고 다음 챕터로 넘어간다 |
| **Surrogate Confession** | 이 장르에서는 음식·습관·루틴이 고백의 매개. 밥을 차려주는 것, 우산을 기다리는 것 |
| **Interior Monologue** | 일상의 반복 속에서 균열이 시작되는 순간. 내면 독백이 흐트러지기 시작할 때 감정이 온다 |

### ✦ AI 프롬프트 (복사해서 바로 사용)

```
문체 기준: 하세쿠라 이스나 대화 구조 + 김애란 감각 묘사
시점: 1인칭 주인공
뼈대 기법: Show Don't Tell + Free Indirect Discourse (고정)
씬별 기법 추가 (1~2개 선택):
  → 감정 고조: Somatization / Interior Monologue
  → 인물 소개: Telling Detail / Sensory Anchoring
  → 긴장 대화: Strategic Silence / Unreliable Narrator lite
  → 챕터 종결: Resonant Closure
클라이맥스: Show Don't Tell + Strategic Silence — 행동 하나로 모든 것을 말한다
금지: 감정 직접 서술, 과잉 수식어, 설명적 서술
```

## 02. 동양 판타지 로맨스
### *Oriental Fantasy Romance — 고아함과 속도감 사이에서 피어나는 감정*

> **주요 플랫폼·레퍼런스:** 카카오페이지 로판 / 네이버시리즈 동양판타지
> **뼈대 기법 (고정):** `Strategic Omission + Pathetic Fallacy`

### 기준 작가 & 문체 레퍼런스

**하세쿠라 이스나** — 「늑대와 향신료」
세계관이 아무리 커도 두 인물 사이의 작은 교환이 이야기의 중심. '눈빛 하나로 다 말하기'.

**중국 선협/고언체 (古言体)** — 「화천골」 「삼생삼세 십리도화」
반복과 대구로 운율. 자연물(달·매화·눈·강)이 감정의 거울. 사자성어는 리듬 포인트로만.

**김훈** — 「칼의 노래」 「남한산성」
문장 하나가 폭력과 슬픔을 동시에 담는다. 꾸밈 없는 동사들의 건조한 아름다움.

> **예시 문단**
>
> 눈이 내렸다. 그는 아무 말도 하지 않았다.
> 그녀가 매화 한 가지를 꺾어 내밀었다.
> "받지 않겠습니다." "버리겠습니까."
> 그는 받았다. 그날 밤 그는 그것을 물에 띄우지 않았다.
> — 말하지 않고 모든 것을 말하는 동양 판타지식 고백 구조

### 문체 뼈대 원칙

- 시점: 3인칭 제한 시점
- 어조: 현대어 기반 + 한자어·자연 이미지 어감 혼합. 완전 고어 금지
- 배경: 계절과 날씨가 인물의 감정 상태를 선행해 묘사
- 대화: 간결하고 격식. 공식/비공식 상황의 낙차가 관계 변화 신호
- 호칭: 공자→그대→이름 변화가 관계 타임라인이자 최강 클라이맥스

### 소주제 — 기법 목록

**◆ 뼈대 기법 (씬마다 반드시 1개 선택, 전체 작품에서 1~2개 고정)**

| 기법 명칭 | 이 장르에서의 작동 방식 |
|-----------|----------------------|
| **Strategic Omission** | 장면을 끝까지 묘사하지 않는다. 독자가 스스로 완성하는 여백이 고아한 감정의 본체 |
| **Pathetic Fallacy** | 자연물이 인물보다 먼저 감정을 말한다. 매화가 지면 관계가 흔들린다는 설계를 처음부터 심는다 |
| **Gaze as Event** | 시선이 대사를 대신한다. 격식 있는 자리에서 눈이 한 번 머무는 것이 이 장르의 고백이다 |

**◆ 살 붙이는 기법 (씬 목적에 따라 1~2개 선택)**

| 기법 명칭 | 이 장르에서의 작동 방식 |
|-----------|----------------------|
| **Title-to-Name Arc** | 호칭 변화가 이 장르의 감정 타임라인. 이름을 처음 부르는 씬을 작품 전체 클라이맥스로 설계 |
| **Surrogate Confession** | 이 장르에서는 사물이 고백 매개. 차를 직접 따라주기, 상처를 먼저 확인하기, 이름 새긴 물건 건네기 |
| **Parallelism** | 대구 구조가 운율을 만든다. 계절·시간·행동을 병렬로 배치해 감정이 쌓이는 리듬을 설계 |
| **Iceberg Theory** | 세계관과 과거의 90%는 독자에게 보이지 않는다. 인물의 반응과 태도로만 유추하게 한다 |
| **Synesthetic Setting** | 시각 장면에 청각·후각을 교차. '달빛'에 '거문고 소리'와 '냉기'를 함께 쓴다 |
| **Register Contrast** | 공식 석상의 냉담함과 단둘일 때의 균열. 이 장르에서는 격식 언어 vs 반말 한마디의 낙차가 폭발력이 된다 |
| **Defamiliarization** | 익숙한 자연물을 낯선 형용사로. '흰 눈' 대신 '소금처럼 내리는 것' |
| **Symbolism** | 자연물·색·계절이 관계를 상징. 처음부터 설계해야 결말에서 폭발력이 생긴다 |
| **Dramatic Irony** | 독자는 인물의 운명을 알지만 주인공은 모른다. 행복한 장면일수록 독자의 심박수가 올라간다 |

### ✦ AI 프롬프트 (복사해서 바로 사용)

```
문체 기준: 선협 고언체 운율 + 김훈식 건조 동사 중심 서술
시점: 3인칭 제한 시점
뼈대 기법: Strategic Omission + Pathetic Fallacy (고정)
씬별 기법 추가 (1~2개 선택):
  → 관계 진전: Title-to-Name Arc / Gaze as Event
  → 감정 전달: Surrogate Confession / Symbolism
  → 세계관: Iceberg Theory / Synesthetic Setting
  → 긴장: Register Contrast / Dramatic Irony
클라이맥스: Strategic Omission + Gaze as Event — 시선 하나로 끊는다. 설명 없음
금지: 완전 고어체, 감정 직접 서술, 세계관 직접 설명
```

## 03. 느와르 로맨스
### *Noir Romance — 어둠 속에서만 보이는 감정의 온도*

> **주요 플랫폼·레퍼런스:** 리디 로맨스 / 카카오페이지 성인로맨스
> **뼈대 기법 (고정):** `Iceberg Theory + Strategic Silence`

### 기준 작가 & 문체 레퍼런스

**레이먼드 챈들러 (Raymond Chandler)** — 「The Big Sleep」 「The Long Goodbye」
문장이 짧고 단호하다. 형용사를 아끼고 꼭 필요한 자리에 낯설고 정확한 것으로만. 냉소적 1인칭 화자가 세계를 관찰하는 동안 감정은 모든 문장에 배어있다.

**김훈** — 「칼의 노래」 「남한산성」
문장 하나가 폭력과 슬픔을 동시에 담는다. 꾸밈 없는 동사들의 무게감. 과잉 없이 묵직하다.

**어니스트 헤밍웨이 (Iceberg Theory)** — 「A Farewell to Arms」
텍스트에 드러난 것은 10%. 말하지 않는 것이 말하는 것보다 강하다.

> **예시 문단**
>
> "나가."
> 그는 대답하지 않았다. 담뱃재를 털었다.
> 나는 다시 말하지 않았다. 그도 가지 않았다.
> 창밖에 비가 왔다.
> — 헤밍웨이식 빙산: 말하지 않는 것이 이야기다

### 문체 뼈대 원칙

- 시점: 1인칭 냉소적 관찰자 (챈들러식) 또는 3인칭 제한
- 문장 호흡: 짧고 단호한 문장. 걸리적거리는 리듬이 오히려 질감
- 형용사: 극도로 절제. 쓸 때는 반드시 낯설고 정확한 것으로
- 대화: 짧고 여백 많다. 말 뒤의 행동이 실제 대사
- 로맨스: 낭만으로 쓰지 않는다. 위험하고 소진되고, 그래도 붙어있는 관계

### 소주제 — 기법 목록

**◆ 뼈대 기법 (씬마다 반드시 1개 선택, 전체 작품에서 1~2개 고정)**

| 기법 명칭 | 이 장르에서의 작동 방식 |
|-----------|----------------------|
| **Iceberg Theory** | 인물의 과거·동기·감정을 설명하지 않는다. 행동과 말투에서만 유추되도록 설계 |
| **Strategic Silence** | 이 장르에서 침묵은 위험의 신호이자 신뢰의 신호. 두 의미가 공존할 때 가장 강하다 |
| **Cynical Gaze** | 화자가 세계를 거리두고 관찰한다. 냉소의 균열이 생기는 지점이 로맨스의 진입점 |

**◆ 살 붙이는 기법 (씬 목적에 따라 1~2개 선택)**

| 기법 명칭 | 이 장르에서의 작동 방식 |
|-----------|----------------------|
| **Embodied Exhaustion** | 감정 대신 신체 소진. 눈물 없이 어깨가 무너진다. 피로가 슬픔이고 통증이 분노다 |
| **Defamiliarization** | 느와르의 도시·실내를 낯선 형용사로. '어두운' 대신 '납처럼 무거운' 빛 |
| **Surrogate Confession** | 이 장르에서 고백은 목숨을 지키는 행동. 먼저 빠져나갈 길을 만들어주기, 이름을 묻지 않기 |
| **Hard Cut** | 감정 정점에서 끊는다. 독자가 채운다. 설명이 없을수록 감정이 남는다 |
| **Noir Atmosphere** | 빛의 방향·그림자의 위치가 긴장을 설계한다. 창문 빛, 가로등, 형광등 하나의 위치가 씬의 감정을 결정 |
| **Narrative Delay** | 이름·신원·과거를 늦게 공개. 독자가 인물을 행동으로만 판단하게 만든다 |
| **Paradoxical Tenderness** | 위험한 인물의 돌발적 부드러움. 이 낙차가 느와르 로맨스의 핵심 감정 폭발 지점 |
| **Contrast** | 이 장르에서 대조는 폭력·냉담 ↔ 단 한순간의 인간적 온기. 그 간격이 좁을수록 강하다 |
| **Dramatic Irony** | 독자는 결말(파국·배신·죽음)을 예감하지만 두 인물은 모른다. 행복한 씬이 가장 위험하다 |

### ✦ AI 프롬프트 (복사해서 바로 사용)

```
문체 기준: 레이먼드 챈들러 하드보일드 + 김훈 건조체 + 헤밍웨이 빙산
시점: 1인칭 냉소적 관찰자
뼈대 기법: Iceberg Theory + Strategic Silence (고정)
씬별 기법 추가 (1~2개 선택):
  → 감정 표현: Embodied Exhaustion / Paradoxical Tenderness
  → 분위기: Noir Atmosphere / Defamiliarization
  → 긴장: Narrative Delay / Dramatic Irony
  → 종결: Hard Cut
클라이맥스: Hard Cut + Embodied Exhaustion — 신체 반응 하나로 끊는다. 눈물도 고백도 없다
금지: 직접 감정 고백, 과잉 수식어, 아름다운 결말 묘사
```

## 04. 로맨스 판타지
### *Fantasy Romance — 세계가 넓을수록 두 사람 사이가 좁아진다*

> **주요 플랫폼·레퍼런스:** 네이버시리즈 로판 / 카카오페이지 로판 (매출 1위 장르)
> **뼈대 기법 (고정):** `In Medias Res + Sensory Anchoring`

### 기준 작가 & 문체 레퍼런스

**Sarah J. Maas** — ACOTAR 시리즈
감각을 과잉에 가까울 정도로 구체화하면서도 독자가 과잉이라고 느끼기 전에 긴장 장면으로 전환. 세계관과 로맨스가 서로를 강화.

**Robin Hobb** — 「피츠버드 연대기」
3인칭이지만 주인공 내면이 너무 깊어 독자는 1인칭처럼 읽는다. 세계관 설명이 모두 주인공의 감정 반응을 통해 전달.

**한국 웹소설 문법** — 「황후님 납시오」 「내 남편과 결혼해줘」
전개 속도 빠름. 매 챕터 클리프행어 필수. 주인공 독백이 코믹하거나 날카롭게 세계관 설명을 대신.

> **예시 문단**
>
> 마법이 손끝에서 터질 것 같았다. 그는 여전히 나를 보고 있었다.
> 전쟁이 내일 시작될 것이었다. 그런데 나는 그의 손이 따뜻하다는 것만 생각했다.
> — Maas식: 세계의 위기와 감정의 위기를 같은 무게로 병치

### 문체 뼈대 원칙

- 시점: 1인칭 (웹소설 몰입도) 또는 3인칭 제한 시점
- 장면 구조: 감각 진입 → 상황 파악 → 내면 반응 → 대화 → 낙차/여운
- 문장 호흡: 짧은 문장 3개 + 긴 문장 1개 / 감정 고조 시 점점 짧게
- 세계관: 드리핑 방식 — 주인공 반응·대화·행동에 녹여 흘리기
- 챕터 끝: 반드시 클리프행어. 독자가 다음 편을 열게 만드는 갈고리

### 소주제 — 기법 목록

**◆ 뼈대 기법 (씬마다 반드시 1개 선택, 전체 작품에서 1~2개 고정)**

| 기법 명칭 | 이 장르에서의 작동 방식 |
|-----------|----------------------|
| **In Medias Res** | 세계관 설명 없이 사건 한가운데로. 독자가 따라오게 만드는 감각과 긴장이 설명을 대체한다 |
| **Sensory Anchoring** | 판타지 세계의 이물감을 오감으로 진입시킨다. 독자가 세계를 '경험'하게 만드는 첫 문장이 핵심 |
| **Free Indirect Discourse** | 세계관 설명이 주인공의 반응·생각·감각으로 자연스럽게 흘러나온다 |

**◆ 살 붙이는 기법 (씬 목적에 따라 1~2개 선택)**

| 기법 명칭 | 이 장르에서의 작동 방식 |
|-----------|----------------------|
| **Drip-feeding Exposition** | 세계관을 행동·대화·실수에 녹인다. "여기서는 그런 말을 하면 안 된다"가 설정 설명보다 강하다 |
| **Emotional Whiplash** | 로맨스 절정 직후 세계의 위기가 온다. 세계관 갈등과 감정 갈등을 같은 무게로 병치 |
| **Cliffhanger** | 이 장르에서 클리프행어는 협상 불가. 고백 직전·신원 폭로·새로운 위협 중 하나로 챕터를 끊는다 |
| **Push-Pull Dynamic** | 세계관의 적대 구조가 관계의 밀당 구조와 맞물려야 한다. 둘이 붙을 수 없는 이유가 서사에 있어야 한다 |
| **Register Contrast** | 공식 자리(전장·궁정·계약)의 냉담함과 단둘일 때의 균열. 이 낙차가 독자 몰입도를 결정한다 |
| **Chekhov's Gun** | 세계관 설정 하나가 클라이맥스에서 결정적으로 작동한다. 처음부터 무심히 심어야 한다 |
| **Unreliable Narrator lite** | 주인공이 상대의 감정을 오독한다. 독자는 정답을 알지만 주인공은 끝까지 부정한다 |
| **Narrative Delay** | 신원·비밀·과거를 늦게 공개. 독자가 알고 싶어서 페이지를 넘기게 만드는 동력 |
| **Synesthetic Setting** | 판타지 세계의 감각을 교차 묘사. '마법의 냄새', '결계의 촉감'처럼 세계관이 감각으로 체험된다 |
| **Dramatic Irony** | 독자는 주인공의 운명·정체·과거를 알지만 주인공은 모른다. 행복한 장면이 가장 위험하다 |

### ✦ AI 프롬프트 (복사해서 바로 사용)

```
문체 기준: Sarah J. Maas 감각 과잉 + Robin Hobb 내면 깊이 + 웹소설 속도감
시점: 1인칭 (기본) / 복잡한 세계관이면 3인칭 제한
뼈대 기법: In Medias Res + Sensory Anchoring (고정)
씬별 기법 추가 (1~2개 선택):
  → 세계관: Drip-feeding / Chekhov's Gun
  → 감정: Free Indirect Discourse / Unreliable Narrator lite
  → 긴장: Emotional Whiplash / Push-Pull / Register Contrast
  → 챕터 종결: Cliffhanger (필수)
클라이맥스: Sensory Anchoring + Hard Cut — 감각 하나에서 끊는다. 세계가 멈추는 순간
금지: 세계관 직접 설명, 감정 직접 서술, 챕터 평온 종결
```

## 05. 회귀·빙의 로맨스
### *Regression / Possession Romance — 두 번째 삶에서도 끝내 그 사람이었다*

> **주요 플랫폼·레퍼런스:** 카카오페이지·네이버시리즈 최강 장르 / 「내 남편과 결혼해줘」 「재혼 황후」
> **뼈대 기법 (고정):** `Dramatic Irony + Narrative Delay`

### 기준 작가 & 문체 레퍼런스

**싱숑 (장성 + 피D)** — 「전지적 독자 시점」
독자만 아는 미래 정보를 최대한 활용한 극적 아이러니. 주인공의 메타적 인식이 서사의 핵심 동력.

**한국 웹소설 회빙환 클리셰 해체** — 「내 남편과 결혼해줘」 「악역의 엔딩은 죽음뿐」
전생 기억을 가진 주인공의 냉정한 계산과 감정적 균열 사이의 긴장감. 독자가 주인공보다 늦게 감정을 인정하게 설계.

> **예시 문단**
>
> 나는 이 사람이 결국 나를 버린다는 걸 알고 있었다.
> 그러니까 웃어서는 안 됐다. 흔들려서도 안 됐다.
> 그런데 그가 내 이름을 불렀다.
> 이번 생에서도. 처음으로.
> — 회귀물의 핵심: 알면서도 무너지는 감정

### 문체 뼈대 원칙

- 시점: 1인칭 주인공 — 전생 기억과 현재 감정의 충돌이 서술의 핵심
- 독자 우월 구조: 독자는 알지만 주변 인물은 모르는 정보 구조 설계
- 기억과 현재의 대비: 전생의 냉담함 ↔ 현재의 균열
- 감정 지연: '이번 생은 다르게 살겠다'는 결심이 감정에 의해 무너지는 과정
- 복선: 전생의 작은 디테일이 현생에서 결정적으로 작동

### 소주제 — 기법 목록

**◆ 뼈대 기법 (씬마다 반드시 1개 선택, 전체 작품에서 1~2개 고정)**

| 기법 명칭 | 이 장르에서의 작동 방식 |
|-----------|----------------------|
| **Dramatic Irony** | 주인공은 전생을 알고, 독자는 그것을 안다. 주변 인물만 모른다. 이 3중 구조가 이 장르의 엔진 |
| **Narrative Delay** | 전생에서 무슨 일이 있었는지를 늦게 공개. 독자가 조각을 모으게 만든다 |
| **Unreliable Narrator lite** | '이번 생은 감정 없이'라는 결심이 서술을 지배하다가 균열이 온다. 독자는 그 균열을 먼저 본다 |

**◆ 살 붙이는 기법 (씬 목적에 따라 1~2개 선택)**

| 기법 명칭 | 이 장르에서의 작동 방식 |
|-----------|----------------------|
| **Chekhov's Gun** | 전생의 소품·말·행동이 현생에서 결정적 의미를 가진다. 독자가 '아' 하는 순간을 설계 |
| **Contrast** | 전생의 장면과 현생의 장면을 병치. 같은 상황, 다른 선택. 그 간격이 성장이고 감정이다 |
| **Free Indirect Discourse** | 전생 기억과 현재 감각이 서술에서 뒤섞인다. 어디가 기억이고 어디가 현재인지 경계가 흐릿해야 한다 |
| **Cliffhanger** | 전생과 다른 선택의 순간, 새로운 변수 등장. 이번 생도 같은 결말로 가는가가 클리프행어의 핵심 |
| **Emotional Whiplash** | 전생 기억이 현재 행복을 침범한다. 행복한 장면에 전생의 파국이 겹쳐 보이는 순간 |
| **Interior Monologue** | 전생 기억 vs 현재 감각의 충돌. 내면 독백이 분열되기 시작할 때 감정이 가장 날것으로 드러난다 |
| **Surrogate Confession** | 전생을 아는 주인공만 할 수 있는 행동. 상대가 이유를 모르는 선물·보호·희생이 고백이 된다 |
| **Anaphora** | '이번 생에서는 —' '그래도 이번에는 —' 반복 구조로 결심과 균열이 쌓인다 |
| **Time Compression / Expansion** | 전생 기억은 압축(순식간에 스친다), 현재 감정 씬은 확장. 두 시간의 밀도 차이가 감정의 무게를 결정 |

### ✦ AI 프롬프트 (복사해서 바로 사용)

```
문체 기준: 회빙환 클리셰 해체 + 감정 지연 설계
시점: 1인칭 주인공 (전생 기억 보유자)
뼈대 기법: Dramatic Irony + Narrative Delay (고정)
씬별 기법 추가 (1~2개 선택):
  → 기억 충돌: Contrast / Interior Monologue
  → 감정 균열: Unreliable Narrator lite / Anaphora
  → 복선 설계: Chekhov's Gun / Time Compression
  → 긴장: Emotional Whiplash / Cliffhanger
클라이맥스: Unreliable Narrator lite 붕괴 + Hard Cut — 부정이 깨지는 순간, 문장이 끊긴다
금지: 전생 기억 직접 설명, 감정 빠른 인정, 챕터 평온 종결
```

## 06. 아포칼립스 로맨스
### *Apocalypse Romance — 세계가 끝나는 날에도 손을 잡았다*

> **주요 플랫폼·레퍼런스:** 카카오페이지 현대판타지 / 문피아 / 「전지적 독자 시점」계열
> **뼈대 기법 (고정):** `In Medias Res + Hard Cut`

### 기준 작가 & 문체 레퍼런스

**코맥 매카시 (Cormac McCarthy)** — 「The Road」
극도로 건조한 문장, 구두점 최소화. 절망 속 인간의 감정이 더 강렬하게 부각된다. 아포칼립스와 부성애·사랑의 교차가 이 장르의 원형.

**싱숑** — 「전지적 독자 시점」
아포칼립스 세계관 + 성좌물 + 로맨스의 결합. 세계의 붕괴를 배경으로 관계의 온도가 점점 올라가는 구조.

> **예시 문단**
>
> 세상이 끝난 지 87일째였다.
> 그는 아직도 내 이름을 정확하게 발음했다.
> 다른 건 다 잃었는데. 그것만은.
> — 아포칼립스 로맨스: 붕괴 속의 온기

### 문체 뼈대 원칙

- 시점: 1인칭 생존자 — 모든 것을 잃은 상태에서 한 사람만 남는 감각
- 문장: 짧고 직관적. 생존 압박이 문장 길이에 반영
- 감정: 극한 상황 속 작은 온기 — 이 대비가 이 장르 로맨스의 핵심
- 세계관: 파괴된 도시·변이된 생물·시스템 붕괴를 구체적 감각으로
- 로맨스: 살기 위해 함께 하다가, 살고 싶어지는 감정으로 진행

### 소주제 — 기법 목록

**◆ 뼈대 기법 (씬마다 반드시 1개 선택, 전체 작품에서 1~2개 고정)**

| 기법 명칭 | 이 장르에서의 작동 방식 |
|-----------|----------------------|
| **In Medias Res** | 붕괴된 세계를 설명하지 않고 그 한가운데로 진입. 세계의 상태가 감각으로 먼저 전달된다 |
| **Hard Cut** | 감정 정점에서 끊는다. 생존 상황의 긴박감과 맞물려 독자가 숨을 참게 만든다 |
| **Sensory Anchoring** | 붕괴된 세계의 감각(녹슨 냄새·재·침묵)이 일상 감각과 대비될 때 온기가 더 강해진다 |

**◆ 살 붙이는 기법 (씬 목적에 따라 1~2개 선택)**

| 기법 명칭 | 이 장르에서의 작동 방식 |
|-----------|----------------------|
| **Iceberg Theory** | 세계가 어떻게 끝났는지, 인물의 전 삶을 설명하지 않는다. 파편적 정보만으로 독자가 재구성한다 |
| **Emotional Whiplash** | 생존 위기 직후의 짧은 온기. 죽을 뻔했다는 것과 살아있다는 것이 동시에 밀려오는 순간 |
| **Cliffhanger** | 생존 위기·동료 배신·세계 붕괴의 새 국면. 아포칼립스에서 클리프행어는 글자 그대로 절벽이다 |
| **Contrast** | 종말 이전 기억 ↔ 현재 폐허. 그리고 그 폐허 속 단 한 사람의 온기. 이 3중 대조가 이 장르의 감정 구조 |
| **Somatization** | 생존 상황에서 감정은 신체로만 드러난다. 배고픔·통증·체온이 감정 신호가 된다 |
| **Surrogate Confession** | 이 장르에서 고백은 생존 행동. 먼저 먹이기, 상처 확인하기, 혼자 보초 서기 |
| **Time Compression / Expansion** | 생존 이동 씬은 압축(며칠이 한 문장), 감정 씬은 확장(2분이 1페이지). 이 밀도 차이가 온기를 강조 |
| **Symbolic Setting** | 배경이 관계를 말한다. 지하 벙커=두 사람만의 세계, 탁 트인 폐허=상실과 자유, 불빛=생존 의지 |
| **Defamiliarization** | 종말 이후 세계를 낯선 형용사로. '조용하다' → '귀가 먹먹할 정도로 비어있다' |

뼈대 기법: In Medias Res + Hard Cut (고정)

  → 세계 묘사: Sensory Anchoring / Defamiliarization

  → 긴장: Emotional Whiplash / Cliffhanger
  → 시간: Time Compression (이동) / Expansion (온기 씬)
클라이맥스: Sensory Anchoring + Hard Cut — 감각 하나(체온·손·숨소리)에서 끊는다

```

## 07. 힐링·순애 로맨스
### *Healing / Pure Love Romance — 상처 입은 사람이 다시 따뜻해지는 이야기*

> **주요 플랫폼·레퍼런스:** 네이버시리즈 현로 / 카카오페이지 힐링물 / 「괴물 공작가의 계약 공녀」계열
> **뼈대 기법 (고정):** `Show Don't Tell + Pathetic Fallacy`

### 기준 작가 & 문체 레퍼런스

**요시모토 바나나 (吉本ばなな)** — 「キッチン(키친)」
상실과 치유를 음식·빛·체온의 감각으로 표현. 슬픔을 밀어내지 않고 함께 앉아있는 문체.

**무라카미 하루키 초기작 스타일** — 「노르웨이의 숲」
조용하고 균일한 문체. 감정이 갑작스럽게 분출되지 않고 서서히 스며든다.

> **예시 문단**
>
> 그는 내가 울고 있다는 걸 알면서도 아무 말 하지 않았다.
> 대신 수프를 한 번 더 저었다.
> 나는 그것으로 충분했다.
> — 힐링 로맨스: 말하지 않아도 알아주는 것

### 문체 뼈대 원칙

- 시점: 1인칭 또는 3인칭 제한
- 문장 호흡: 느리고 고요하다. 서두르지 않는다
- 감정: 작은 것들 — 따뜻한 수프, 조용한 웃음, 창밖 빛 — 이 치유의 언어
- 갈등: 트라우마·상처·두려움. 사랑을 막는 것이 외부가 아닌 내면
- 로맨스: 극적 이벤트 없이, 함께 있는 시간이 쌓여 사랑이 되는 구조

### 소주제 — 기법 목록

**◆ 뼈대 기법 (씬마다 반드시 1개 선택, 전체 작품에서 1~2개 고정)**

| 기법 명칭 | 이 장르에서의 작동 방식 |
|-----------|----------------------|
| **Show Don't Tell** | 치유는 설명하지 않는다. 밥을 먹고, 잠을 자고, 처음으로 웃는 것. 그것이 회복의 전부다 |
| **Pathetic Fallacy** | 계절이 인물의 회복 속도를 말한다. 겨울이 길수록 봄이 강하다. 환경이 치유의 타임라인 |
| **Sensory Anchoring** | 이 장르의 감각은 온기·부드러움·익숙한 냄새. 위협이 아닌 안전감을 여는 감각 |

**◆ 살 붙이는 기법 (씬 목적에 따라 1~2개 선택)**

| 기법 명칭 | 이 장르에서의 작동 방식 |
|-----------|----------------------|
| **Surrogate Confession** | 이 장르에서 고백은 돌봄. 상처를 먼저 발견하기, 좋아하는 것을 기억했다가 준비하기 |
| **Telling Detail** | 인물의 상처를 소품으로 드러낸다. 해진 소매, 먹다 남긴 밥, 잠글 수 없는 문 |
| **Strategic Silence** | 말하지 않아도 이해받는 침묵. 이 장르에서 침묵은 위협이 아닌 안전의 신호 |
| **Somatization** | 회복의 신체화. 처음으로 어깨가 내려앉는 것, 손에서 힘이 빠지는 것이 치유의 신호 |
| **Symbolism** | 치유의 상징을 처음부터 심는다. 빛·음식·계절·식물이 관계 회복의 타임라인을 대신한다 |
| **Interior Monologue** | 상처 입은 내면의 방어기제가 허물어지는 과정. 독백이 조각조각 무너지기 시작할 때 치유가 온다 |
| **Time Compression / Expansion** | 회복은 느리다(압축 없이 천천히 흐른다). 단 한순간의 따뜻함은 확장된다 |
| **Resonant Closure** | 완전히 닫지 않는다. 치유는 끝나지 않는다. 그래도 조금 나아졌다는 여운 |

### ✦ AI 프롬프트 (복사해서 바로 사용)

```
문체 기준: 요시모토 바나나 치유 감각 + 하루키 균일한 호흡
시점: 1인칭
뼈대 기법: Show Don't Tell + Pathetic Fallacy (고정)
씬별 기법 추가 (1~2개 선택):
  → 치유 씬: Surrogate Confession / Sensory Anchoring
  → 내면: Interior Monologue / Somatization
  → 상징: Symbolism / Telling Detail
  → 종결: Resonant Closure
클라이맥스: Show Don't Tell + Strategic Silence — 처음으로 손을 내밀거나, 처음으로 울거나. 행동 하나. 설명 없음
금지: 극적 갈등 과잉, 트라우마 설명적 서술, 빠른 감정 해결
```

## 08. 공포·호러 로맨스
### *Horror Romance — 두려운 것과 사랑에 빠지다*

> **주요 플랫폼·레퍼런스:** 카카오페이지 스릴러·공포 / 리디 다크로맨스
> **뼈대 기법 (고정):** `Sensory Anchoring + Strategic Omission`

### 기준 작가 & 문체 레퍼런스

**셜리 잭슨 (Shirley Jackson)** — 「The Haunting of Hill House」
공포는 외부에 있지 않다. 화자의 내면에서 자라나는 불안. 서술 자체가 불안정해지는 방식으로 공포를 전달.

**피아 와츠 (PIA WATTS) 스타일 다크로맨스** — 「Haunting Adeline」계열
괴물과 인간의 경계, 위험한 존재에 대한 끌림. 공포와 에로스가 교차하는 지점이 이 장르의 핵심.

> **예시 문단**
>
> 그가 웃었다. 눈은 웃지 않았다.
> 나는 한 발 물러섰다. 한 발 다가갔다.
> 어쩌면 그게 같은 움직임이었는지도 모른다.
> — 공포 로맨스: 끌림과 두려움은 같은 방향

### 문체 뼈대 원칙

- 시점: 1인칭 — 독자가 화자의 불안을 함께 느끼도록
- 문장: 표면은 평온하지만 서서히 균열이 보인다
- 공포: 설명하지 않는다. 무언가 이상하다는 감각이 먼저
- 로맨스: 위험한 존재에 대한 끌림. '도망쳐야 하는데 가고 싶다'
- 분위기: 빛과 어둠, 소음과 침묵의 대비로 긴장 조성

### 소주제 — 기법 목록

**◆ 뼈대 기법 (씬마다 반드시 1개 선택, 전체 작품에서 1~2개 고정)**

| 기법 명칭 | 이 장르에서의 작동 방식 |
|-----------|----------------------|
| **Sensory Anchoring** | 이 장르의 감각은 이물감·불편함·차가움. 뭔가 이상하다는 신호를 감각으로 먼저 전달한다 |
| **Strategic Omission** | 공포는 보이지 않는 것이다. 설명하지 않는 빈 공간이 독자의 상상을 가장 어두운 곳으로 데려간다 |
| **Unreliable Narrator lite** | 화자가 자신의 공포를 부정한다. '이상한 게 아니야'라고 말할수록 독자는 더 두렵다 |

**◆ 살 붙이는 기법 (씬 목적에 따라 1~2개 선택)**

| 기법 명칭 | 이 장르에서의 작동 방식 |
|-----------|----------------------|
| **Defamiliarization** | 익숙한 공간을 낯선 형용사로. 집이 '집처럼 보이지 않는' 순간이 공포의 시작 |
| **Dramatic Irony** | 독자는 위험을 알지만 화자는 모른다. 화자가 안심할수록 독자의 긴장이 높아진다 |
| **Cliffhanger** | 공포 로맨스의 클리프행어는 정체 폭로 또는 위험 노출. 끌림이 공포로, 공포가 끌림으로 반전 |
| **Contrast** | 일상적 온기(웃음·음식·빛) ↔ 공포적 이물감. 이 낙차가 클수록 독자가 더 깊이 빠진다 |
| **Interior Monologue** | 불안이 합리화되다가 균열이 온다. 내면 독백이 흔들리기 시작할 때 공포가 진입한다 |
| **Symbolism** | 공포의 상징을 반복. 특정 소리·색·냄새가 위험의 신호로 패턴화되면 이후엔 그것만으로 긴장이 온다 |
| **Narrative Delay** | 상대의 정체·목적을 늦게 공개. 알 수 없다는 것 자체가 공포이자 끌림 |
| **Somatization** | 공포와 끌림을 같은 신체 반응으로. 심장이 빨라지는 것이 두려움인지 설레임인지 모르는 상태 |
| **Hard Cut** | 공포 씬의 절정에서 끊는다. 보이지 않는 것이 보인 것보다 무섭다 |

## 09. 새드엔딩·비극 로맨스
### *Tragedy Romance — 사랑했기 때문에, 그래서 끝이 났다*

> **주요 플랫폼·레퍼런스:** 카카오페이지 로맨스 / 리디 / 「낮에 뜨는 달」계열
> **뼈대 기법 (고정):** `Dramatic Irony + Pathetic Fallacy`

### 기준 작가 & 문체 레퍼런스

**알베르 카뮈 (Albert Camus)** — 「이방인」
감정을 직접 서술하지 않는다. 담담한 문장이 오히려 비극의 무게를 극대화한다.

**이청준** — 「당신들의 천국」
사랑이 상처를 주는 구조. 선의가 비극으로 귀결되는 아이러니. 결말을 향해 천천히 조여오는 문체.

> **예시 문단**
>
> 그해 봄은 유독 꽃이 많이 피었다.
> 나는 그것이 나쁜 징조라는 걸 몰랐다.
> 아니, 알고 있었다.
> 그래서 더 오래 봤다.
> — 새드엔딩 로맨스: 예감하면서도 붙잡는 것

### 문체 뼈대 원칙

- 시점: 1인칭 또는 전지적 — 독자가 결말을 예감하게 설계
- 문장: 담담하고 균일하다. 감정을 과잉하면 비극의 무게가 가벼워진다
- 구조: 독자는 비극을 예감하지만 주인공은 모른다 (극적 아이러니)
- 행복 장면: 비극 전에 가장 행복한 장면을 배치 — 낙차가 비극의 강도
- 결말: 감정 해소 없이 끝낸다. 여운이 비극의 본체다

### 소주제 — 기법 목록

**◆ 뼈대 기법 (씬마다 반드시 1개 선택, 전체 작품에서 1~2개 고정)**

| 기법 명칭 | 이 장르에서의 작동 방식 |
|-----------|----------------------|
| **Dramatic Irony** | 독자가 비극의 결말을 예감한다. 주인공이 행복할수록 독자의 슬픔이 먼저 온다 |
| **Pathetic Fallacy** | 자연이 결말을 먼저 말한다. 꽃이 피는 장면에 이별의 냄새가 배어있다 |
| **Anaphora** | 같은 문장 구조의 반복이 비극을 향해 쌓인다. 반복이 많을수록 결말이 불가피해 보인다 |

**◆ 살 붙이는 기법 (씬 목적에 따라 1~2개 선택)**

| 기법 명칭 | 이 장르에서의 작동 방식 |
|-----------|----------------------|
| **Narrative Delay** | 이 장르에서 지연은 비극 공개를 늦추는 것. 독자가 알면서도 조금 더 행복하게 해달라고 바라게 만든다 |
| **Symbolism** | 처음에 배치한 상징이 결말에서 비극의 신호로 반전. 같은 사물이 처음엔 설레임, 끝엔 이별을 말한다 |
| **Contrast** | 가장 행복한 씬 직후 비극. 또는 전작의 행복 ↔ 현재의 상실. 낙차가 클수록 비극이 강하다 |
| **Strategic Silence** | 마지막으로 나눠야 할 말을 하지 않는다. 침묵이 비극의 본체가 된다 |
| **Show Don't Tell** | 슬픔을 설명하지 않는다. 그가 떠난 자리에 컵이 하나 남아있다. 그것이 전부다 |
| **Free Indirect Discourse** | 주인공의 합리화와 슬픔이 서술에 뒤섞인다. 독자는 주인공이 모르는 슬픔을 먼저 읽는다 |
| **Time Compression / Expansion** | 행복한 시간은 압축(빠르게 지나간다), 이별 씬은 확장(느리게 흐른다). 이 비대칭이 비극의 감각 |
| **Resonant Closure** | 완전히 닫지 않는다. 비극이 독자 안에서 계속 살아있게 한다 |
| **Hard Cut** | 이별을 설명하지 않는다. '그는 가지 않았다'가 아니라 '그는 갔다.' 그뿐 |

### ✦ AI 프롬프트 (복사해서 바로 사용)

```
문체 기준: 카뮈 담담체 + 이청준 조여오는 구조
시점: 1인칭 또는 전지적 제한
뼈대 기법: Dramatic Irony + Pathetic Fallacy (고정)
씬별 기법 추가 (1~2개 선택):
  → 예감: Symbolism / Anaphora
  → 감정: Free Indirect Discourse / Contrast
  → 긴장: Narrative Delay / Strategic Silence
  → 종결: Resonant Closure 또는 Hard Cut (택 1, 혼용 금지)
클라이맥스: Show Don't Tell + Hard Cut — 감정 없이 사실만 남긴다. 독자가 무너진다
금지: 감정 과잉, 눈물 직접 묘사, 해피엔딩 암시, 설명적 서술
```

## 10. 현대판타지 로맨스
### *Modern Fantasy Romance — 평범한 현실에 균열이 생기자, 그 사람이 보였다*

> **주요 플랫폼·레퍼런스:** 카카오페이지 현판 / 네이버시리즈 / 「나 혼자만 레벨업」계열
> **뼈대 기법 (고정):** `In Medias Res + Drip-feeding Exposition`

### 기준 작가 & 문체 레퍼런스

**나루토·헌터×헌터 작가 문법 + 한국 웹소설 속도감** — 「나 혼자만 레벨업」 「전직 검사」
현실 세계에 판타지 시스템이 삽입. 주인공의 성장이 로맨스와 교차. 빠른 전개와 시스템 설명을 독백으로 처리.

**기예르모 델 토로 (Guillermo del Toro) 감성** — 「판의 미로」
현실과 판타지의 경계가 모호한 세계관. 공포와 아름다움이 교차. 현대판타지 로맨스의 감성적 레퍼런스.

> **예시 문단**
>
> [스킬: 마음 읽기 - Lv.1] 이 알림이 뜬 건 이번이 세 번째였다.
> 첫 번째는 내가 죽을 뻔한 날.
> 두 번째는 그가 나를 구한 날.
> 세 번째는 오늘, 지금.
> — 현대판타지 로맨스: 시스템 언어로 감정을 코딩

### 문체 뼈대 원칙

- 시점: 1인칭 — 시스템·HUD 묘사를 독백에 자연스럽게 녹임
- 속도감: 웹소설 문법 최적. 챕터당 정보 밀도 높지만 가독성 유지
- 세계관: 현실 일상 + 이물감(던전·게이트·시스템) 대비를 감각으로
- 로맨스: 생사를 함께한 관계 → 서로를 지키고 싶은 감정으로 자연 발전
- 강함의 미학: 주인공의 성장이 로맨스 파트너에게 인정받는 과정이 이 장르의 고백

### 소주제 — 기법 목록

**◆ 뼈대 기법 (씬마다 반드시 1개 선택, 전체 작품에서 1~2개 고정)**

| 기법 명칭 | 이 장르에서의 작동 방식 |
|-----------|----------------------|
| **In Medias Res** | 현실에 균열이 생긴 순간으로 바로 진입. 설명 없이 위기로 시작, 세계관은 반응으로 드러난다 |
| **Drip-feeding Exposition** | 시스템 설명을 주인공의 반응·독백·실수에 녹인다. 튜토리얼이 아닌 생존 경험으로 세계관을 전달 |
| **Free Indirect Discourse** | 시스템 알림·스킬 정보가 주인공의 생각·감각과 섞인다. HUD가 감정 언어가 되는 지점 |

**◆ 살 붙이는 기법 (씬 목적에 따라 1~2개 선택)**

| 기법 명칭 | 이 장르에서의 작동 방식 |
|-----------|----------------------|
| **Cliffhanger** | 이 장르에서 클리프행어는 필수. 던전 돌입·레이드 실패·시스템 오류가 감정 클리프행어와 동시에 온다 |
| **Emotional Whiplash** | 전투 직후의 생존 감각과 감정이 겹친다. 죽을 뻔했다는 것, 그래도 살았다는 것, 그리고 그 사람 |
| **Show Don't Tell** | 이 장르에서 감정은 전투 패턴으로 드러난다. 그를 위해 스킬을 아끼는 것, 먼저 방어 포지션 잡는 것 |
| **Surrogate Confession** | 이 장르의 고백은 전투 지원. 포션 먼저 건네기, 혼자 뛰어들지 않기, 귀환 스킬을 함께 쓰기 |
| **Chekhov's Gun** | 1화의 시스템 오류·약한 스킬·버려진 아이템이 클라이맥스에서 결정적으로 작동한다 |
| **Contrast** | 각성 전의 약함 ↔ 성장 후의 강함. 그리고 강해졌어도 그 사람 앞에서만 흔들리는 것 |
| **Somatization** | 전투 긴장과 감정 긴장을 같은 신체 반응으로. 심장이 빨라지는 것이 몬스터 때문인지 그 때문인지 |
| **Register Contrast** | 레이드 현장의 냉담한 파티 리더 ↔ 단둘이 귀환할 때의 다른 사람. 이 낙차가 이 장르의 설레임 |
| **Time Compression / Expansion** | 전투 씬은 빠르게(압축), 감정 씬은 느리게(확장). 전투 중 한순간의 눈 맞춤이 3페이지가 된다 |

### ✦ AI 프롬프트 (복사해서 바로 사용)

```

시점: 1인칭 (시스템 창·HUD를 독백처럼 자연스럽게)
뼈대 기법: In Medias Res + Drip-feeding (고정)
씬별 기법 추가 (1~2개 선택):
  → 액션: Emotional Whiplash / Cliffhanger
  → 감정: Surrogate Confession / Somatization
  → 세계관: Chekhov's Gun / Contrast

## 11. BL / GL 로맨스
### *Boys Love / Girls Love Romance — 사랑의 형태를 묻지 않는다, 감정이 있을 뿐*

> **주요 플랫폼·레퍼런스:** 카카오페이지 BL / 리디 BL·GL / 문피아
> **뼈대 기법 (고정):** `Free Indirect Discourse + Unreliable Narrator lite`

### 기준 작가 & 문체 레퍼런스

**나기타 코지 (凪良ゆう)** — 「流浪の月(방랑하는 달)」
사회의 시선과 내면의 감정 사이에서 갈등하는 구조. 외부 시선을 의식하는 내면 독백이 이 장르의 핵심 문체 장치.

**마쓰오카 게이스케 (松岡圭祐) 감성 + 한국 BL 웹소설** — 카카오페이지 BL 상위 작품들
감정을 철저히 숨기는 캐릭터가 서서히 무너지는 구조. 감정의 균열이 한 번에 터지는 순간이 이 장르의 클라이맥스.

> **예시 문단**
>
> 이상하다고 생각했다.
> 그의 목소리가 들릴 때마다 등이 곧아졌다. 그게 이상했다.
> 이름을 부르면 심장이 울렸다. 그것도 이상했다.
> 전부 이상했다. 그래서 아무 의미 없었다.
> — BL/GL 로맨스: 부정이 쌓여 감정이 된다

### 문체 뼈대 원칙

- 시점: 1인칭 — 인정하기 싫은 감정과의 싸움이 서술의 본체
- 사회적 시선: 내면 독백 안에 외부 시선이 항상 존재. 이것이 장르의 고유 긴장감
- 감정 지연: '이건 우정이다'라고 부정하다가 무너지는 과정
- 신체화: 눈길, 접촉, 체온 — 작은 신체 접촉이 감정의 폭발 직전까지 쌓임
- 로맨스: 사회 통념과 내면 감정의 충돌이 해소될 때가 이 장르의 클라이맥스

### 소주제 — 기법 목록

**◆ 뼈대 기법 (씬마다 반드시 1개 선택, 전체 작품에서 1~2개 고정)**

| 기법 명칭 | 이 장르에서의 작동 방식 |
|-----------|----------------------|
| **Free Indirect Discourse** | 부정하는 서술자의 목소리와 감정이 새어나오는 서술이 한 문장 안에 공존한다 |
| **Unreliable Narrator lite** | '이건 아무것도 아니야'라는 부정이 서술을 지배한다. 독자는 첫 챕터부터 진실을 안다 |
| **Strategic Silence** | 말하지 못한 것들. 침묵이 이 장르에서는 사회적 억압과 감정의 압력 두 가지를 동시에 담는다 |

**◆ 살 붙이는 기법 (씬 목적에 따라 1~2개 선택)**

| 기법 명칭 | 이 장르에서의 작동 방식 |
|-----------|----------------------|
| **Somatization** | 감정을 인정하기 전 신체가 먼저 반응한다. 등이 곧아지고, 손가락이 굳고, 목이 마른다 |
| **Interior Monologue** | 부정과 감정 사이에서 내면 독백이 분열된다. 합리화가 무너지는 순간이 클라이맥스의 전조 |
| **Anaphora** | '이상하다', '아니다', '그게 다다' — 부정 구조의 반복이 쌓여 결국 역전된다 |
| **Narrative Delay** | 고백을 늦춘다. 독자는 알고 주인공은 부정한다. 이 긴장이 챕터를 넘기게 만드는 동력 |
| **Register Contrast** | 사람들 앞에서의 냉담함 ↔ 단둘일 때 새어나오는 균열. 이 낙차가 이 장르의 핵심 설레임 |
| **Surrogate Confession** | 이 장르에서 고백은 사회적으로 드러나지 않는 행동. 먼저 우산을 기다리기, 이름을 한 박자 늦게 부르기 |
| **Telling Detail** | 상대에게만 보이는 디테일. 다른 사람은 모르는 습관·표정·목소리 톤을 주인공만 안다 |
| **Contrast** | '다른 사람에게 하는 것'과 '나에게만 하는 것'의 대조. 독자가 먼저 그 차이를 발견한다 |
| **Resonant Closure** | 이 장르의 챕터 종결은 여운. 부정이 완전히 무너지지 않은 채로 끊어진 한 문장 |

### ✦ AI 프롬프트 (복사해서 바로 사용)

```
문체 기준: 나기타 코지 내면 갈등 + 한국 BL 웹소설 감정 균열 구조
시점: 1인칭 (사회적 시선이 내면 독백 안에 항상 존재)
뼈대 기법: Free Indirect Discourse + Unreliable Narrator lite (고정)
씬별 기법 추가 (1~2개 선택):
  → 감정 부정: Anaphora / Interior Monologue
  → 신체 반응: Somatization / Telling Detail
  → 관계 진전: Register Contrast / Surrogate Confession
  → 종결: Resonant Closure
클라이맥스: Unreliable Narrator lite 붕괴 + Strategic Silence — 부정이 멈추는 순간. 침묵. 그뿐
금지: 감정 빠른 인정, 사회적 시선 무시한 전개, 설명적 서술
```

# 🔀 씬 목적별 기법 조합 가이드

> 씬의 '감정 목표'를 먼저 정하고, 그에 맞는 기법을 고른다.
> 기법이 씬의 목적을 이끌어야지, 기법을 넣기 위해 씬을 쓰면 안 된다.

| 씬 감정 목표 | 권장 기법 조합 | 주의사항 |
|-------------|--------------|---------|
| 첫 만남·첫 인상 | `Sensory Anchoring + Telling Detail` | 설명 금지. 감각과 디테일만으로 인물을 드러낸다 |
| 감정 인지 (설레임) | `Show Don't Tell + Somatization` | 신체 반응으로만. '설레었다' 절대 금지 |
| 밀당·긴장 | `Push-Pull Dynamic + Register Contrast` | 한쪽이 가까워지면 반드시 다른 쪽이 물러난다 |
| 고백 직전 | `Narrative Delay + Unreliable Narrator lite` | 기법 수 최소화. 리듬과 문장 길이로 긴장 조절 |
| 간접 고백 | `Surrogate Confession + Strategic Silence` | 말하지 않는다. 행동만으로 감정을 드러낸다 |
| 세계관 설명 | `Drip-feeding + Iceberg Theory` | 설명이 아닌 반응. 주인공이 세계에 반응하게 한다 |
| 갈등·대립 | `Emotional Whiplash + Free Indirect Discourse` | 내면과 외부 갈등을 교차시켜 독자를 압박 |
| 배경·분위기 | `Sensory Anchoring + Pathetic Fallacy` | 감각 → 환경 → 감정 순서로 연결 |
| 비극 예감 | `Dramatic Irony + Symbolism` | 독자는 알지만 주인공은 모른다. 상징으로 결말 암시 |
| 치유 씬 | `Surrogate Confession + Sensory Anchoring` | 작은 행동과 감각. 따뜻함을 설명하지 않는다 |
| 클라이맥스 | `Show Don't Tell + Hard Cut` | 기법 최소화. 감정만 남긴다. 설명 금지 |
| 챕터 종결 (웹소설) | `Cliffhanger (필수)` | 독자가 다음 편을 열게 만드는 갈고리. 협상 불가 |
| 챕터 종결 (문학적) | `Resonant Closure` | 툭 잘린 여운 한 문장. 완전히 닫지 않는다 |

### Surrogate Confession 장르별 작동 차이

> 이 기법은 전 장르에서 사용되지만 **고백의 매개**가 장르마다 달라야 한다.

| 장르 | 고백의 매개 | 예시 |
|------|------------|------|

| 동양판타 | 사물·의식·몸짓 | 직접 차를 따르기, 이름 새긴 물건 |
| 느와르 | 생존 보호 행동 | 먼저 빠져나갈 길을 만들어주기 |
| 로판·현판 | 전투 지원 | 스킬·포션·방어 포지션을 먼저 |
| 아포칼립스 | 생존 나눔 | 먹이기, 보초 서기, 이름 정확히 부르기 |
| 힐링 | 돌봄 행동 | 상처 먼저 발견하기, 좋아하는 것 준비하기 |
| BL/GL | 사회적으로 드러나지 않는 행동 | 이름을 한 박자 늦게 부르기, 먼저 우산을 기다리기 |
