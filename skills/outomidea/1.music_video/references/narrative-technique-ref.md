## 1. 기법 코드 속기표 (전체)

| 코드 | 기법명 | 설명 (1줄) |
|------|--------|-----------|
| `[SDT]` | Show Don't Tell | 감정 명명 금지 → 신체/행동/환경으로 |
| `[FID]` | Free Indirect Discourse | 서술자+주인공 목소리 혼합 |
| `[SEN]` | Sensory Anchoring | 오감 중 하나로 씬 진입 |
| `[SIL]` | Strategic Silence | 침묵·행동이 말보다 많은 것을 전달 |
| `[DEL]` | Narrative Delay | 핵심 정보를 의도적으로 늦게 공개 |
| `[PPD]` | Push-Pull Dynamic | 가까워지면 밀고 멀어지면 당기는 구조 |
| `[SUB]` | Surrogate Confession | 행동·사물·제3자를 통한 간접 고백 |
| `[REG]` | Register Contrast | 공식 석상 냉담 ↔ 단둘일 때 균열 |
| `[RES]` | Resonant Closure | 챕터를 완전히 닫지 않는 여운 종결 |
| `[DET]` | Telling Detail | 설명 없이 캐릭터를 드러내는 소품·습관 |
| `[SOM]` | Somatization | 감정 → 특정 신체 부위 반복 연결 |
| `[UNR]` | Unreliable Narrator lite | 주인공이 자기 감정을 스스로 부정 |
| `[PATH]` | Pathetic Fallacy | 자연환경이 인물 감정을 선행 반영 |
| `[EWH]` | Emotional Whiplash | 아름다운 장면 → 갑작스러운 위기 |
| `[CLF]` | Cliffhanger | 반전·미완·위기로 챕터 끊기 |
| `[IMR]` | In Medias Res | 사건 한가운데서 시작 |
| `[DRP]` | Drip-feeding Exposition | 세계관을 조금씩 흘리기 |
| `[ICE]` | Iceberg Theory | 10%만 드러내고 90%는 압축 |
| `[CHK]` | Chekhov's Gun | 복선 소품이 후반에 결정적 작동 |
| `[PAR]` | Parallelism | 같은 구조 반복으로 감정 증폭 |
| `[SYN]` | Synesthetic Setting | 시각+청각+후각을 교차한 공감각 묘사 |
| `[HCT]` | Hard Cut | 감정 설명 없이 문장을 끝냄 |
| `[OMIT]` | Strategic Omission | 장면을 끝까지 묘사하지 않는 여백 |
| `[DEF]` | Defamiliarization | 익숙한 사물을 낯선 형용사로 |
| `[TNA]` | Title-to-Name Arc | 호칭 변화(직책→이름)를 관계 타임라인으로 |
| `[DRI]` | Dramatic Irony | 독자는 알고 주인공은 모르는 구조 |
| `[ANA]` | Anaphora | 문장 첫머리 반복으로 감정 점층 |
| `[TCX]` | Time Compression/Expansion | 감정 씬=확장, 이동 씬=압축 |
| `[INT]` | Interior Monologue | 의식의 흐름 같은 내면 독백 |
| `[SYM]` | Symbolism | 사물·색·날씨를 감정/관계 상징으로 |
| `[CON]` | Contrast | 두 인물·장면·감정의 의도적 병치 |
| `[GAZ]` | Gaze as Event | 시선이 대사·행동을 대신함 |
| `[EMB]` | Embodied Exhaustion | 감정 대신 신체 소진 서술 |

## 5. 공통 규칙

- Show Don't Tell: 감정 명명 금지 → 신체/환경/행동으로만
- 3인칭 제한 시점 (별도 지정 없는 한)
- 모든 묘사 = 복선 or 캐릭터 정보
- 매 턴 최소 1가지 새 정보/사건/감정 변화
- 기법 지시문 줄당 ≤100자

> 상세 정의·예시는 `romance_guide.md` 참조 (파이프라인 로드 필요 없음)
