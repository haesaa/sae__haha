## 2. 감정 상태 분류 (무드 사전 연동)

감정 상태는 SKILL-1A의 공식 무드 사전 24개 태그와 매핑됨:

### Enneagram 분열 방향 → 무드 태그

| Enneagram 분열 감정 | 무드 태그 |
|--------------------|----------|
| 우울, 자기연민 (타입1→4) | `melancholic` |
| 공격적 충동 (타입2→8) | `aggressive` / `angry` |
| 불안, 의존 (타입6→3) | `anxious` / `fearful` |
| 은둔, 냉소 (타입8→5) | `lonely` / `cold` |
| 갈등/분리 공포 (타입9→6) | `fearful` / `tense` |

### 애착 유형 → 감정-환경 매핑

| 애착 상태 | 유발 감정 | 환경 키워드 |
|----------|----------|------------|
| 불안형 활성화 | anxious / fearful | narrow, enclosed |
| 회피형 활성화 | lonely / cold | open, isolated |
| 혼란형 활성화 | confused / unresolved | chaotic, unstable |
| 안정형 | peaceful / warm | natural, open |

## 4. 설계 원칙 (요약)

- **Show Don't Tell**: 감정 명명 금지 → 신체/행동으로 표현
- **유형 간 표면 모순 = 캐릭터 깊이**: 모순이 내적 갈등의 근거
- **[SAFE]/[UNSAFE] 분리** 필수 — 세이프 작품엔 [UNSAFE] 블록 전체 제거
- **랜덤 배정 금지** — 모든 유형은 캐릭터 설정에서 행동 근거 추출 후 연결
