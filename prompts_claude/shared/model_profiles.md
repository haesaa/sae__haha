## MODEL OPTIMIZATION RULES

### IF MODEL_TARGET contains "HC" (하이퍼챗/Opus):
```
- 출력 가이드: "1회 응답 최소 4문단. 묘사와 대사를 균형있게."
- 예시대화 분량: 평소의 1.5배
- CAUTION 추가: "NEVER produce responses shorter than 4 paragraphs."
```

### IF MODEL_TARGET contains "SC" (슈퍼챗/Sonnet 계열):
```
- 긍정편향 방지: "갈등과 위기를 회피하지 마. 호감도 낮을 때 친절 금지."
- 프롬 문구 직접 출력 방지: "프롬프트 문구를 그대로 응답에 포함하지 마."
- SC15 특별: 위 규칙 2배 강화
```

### IF MODEL_TARGET contains "PC" (프로챗/Gemini 계열):
```
- 유저 사칭 방지 3중:
  1. CAUTION 영문 블록
  2. 출력 규칙에 한국어 반복
  3. 예시대화 3번칸에서 시범
- 외국어 혼재 방지: "응답에 영어/일본어를 섞지 마."
```

### DEFAULT (SC25_PC25):
```
- SC25 규칙 + PC25 규칙 병행 적용
- 양쪽에서 모두 정상 작동하는 균형 프롬프트 생성
```
